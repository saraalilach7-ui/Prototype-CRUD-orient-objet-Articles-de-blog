# Blog Manager

## Description
Application PHP pour gérer les articles (CRUD)

## Technologies
- PHP (POO)
- MySQL
- PDO

## Fonctionnalités
- Ajouter un article
- Modifier un article
- Supprimer un article
- Afficher les articles

## Installation
1. Importer script.sql
2. Configurer database.php
3. Lancer le projet sur localhost
